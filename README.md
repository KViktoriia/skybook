# SkyBook — Платформа бронювання авіаквитків ✈️

**SkyBook** — це сучасний, високотехнологічний вебзастосунок (SPA) для пошуку, вибору та онлайн-бронювання авіаквитків. Проєкт розроблено в межах курсу «Технології створення програмних продуктів» (ТСПП).

Додаток поєднує в собі преміальний адаптивний інтерфейс (з використанням скляного морфізму, анімацій та темної теми) і повноцінне імітаційне моделювання бізнес-процесів продажу квитків, включаючи інтеграцію з державними API, платіжними шлюзами та системами безпеки.

---

## 🌟 Ключові можливості

1. **Пошук та фільтрація рейсів**:
   - Пошук рейсів між містами України та міжнародними хабами.
   - Сортування рейсів за ціною (зростання/спадання) та тривалістю перельоту.
   - Динамічний розрахунок дат польотів на 10 днів вперед від поточної дати.
2. **Інтерактивна схема літака**:
   - Візуальний вибір місць на борту Boeing 737 або Airbus A320.
   - Чіткий розподіл на Бізнес-клас (ряди 1-3, компоновка 2-2) та Економ-клас (ряди 4-17, компоновка 3-3).
   - Блокування вже зайнятих місць та динамічний розрахунок ціни (+50% націнка за Бізнес).
3. **Швидкий Onboarding (Дія API)**:
   - Симуляція FaceID авторизації та миттєвого імпорту паспортних даних пасажира через державний сервіс «Дія».
4. **Захищений платіжний шлюз та Фрод-моніторинг**:
   - Маски введення банківських карт (номер, термін дії MM/YY, CVV).
   - Інтегрований аналізатор ризиків (Fraud Scoring). Транзакції за картками або паспортами з чорного списку (наприклад, паспорт `FT111111`) автоматично блокуються.
5. **Генерація Boarding Pass (Посадочного талона)**:
   - Створення електронного квитка з унікальним кодом PNR та реалістичним QR-кодом.
   - Оптимізована друкована версія (CSS Print Stylesheet) для виведення квитка на принтер або збереження в PDF на одну сторінку А4.
6. **Особистий кабінет пасажира**:
   - Перегляд активних квитків та їх статусів.
   - Функція скасування квитка з автоматичним розрахунком суми повернення (з утриманням 20% збору авіакомпанії).
7. **Підвищення прав до Адміністратора**:
   - Можливість увійти в режим адміністратора безпосередньо з Особистого кабінету за допомогою секретного ключа `admin2026`.
   - Керування розкладом польотів (додавання нових рейсів, які миттєво з'являються в пошуку).
   - Жива фінансова статистика (чистий дохід, відсоток заповнюваності, загальна кількість бронювань).

## 💾 Архітектура даних (База даних у браузері)
Для забезпечення мобільності та запуску без стороннього серверного софту, база даних симулюється у **HTML5 LocalStorage** вашого браузера з реляційною структурою:
- `skybook_users` — акаунти користувачів, паролі, ролі (`passenger`/`admin`) та статуси.
- `skybook_flights` — розклад польотів та зайняті місця.
- `skybook_bookings` — зареєстровані квитки пасажирів з PNR кодами.
- `skybook_transactions` — лог фінансових транзакцій (APPROVED / REFUNDED / BLOCKED).
- `skybook_blacklist` — чорний список паспортів та карт для системи безпеки.

## 🚀 Як запустити проєкт локально

### Варіант 1: Швидкий запуск
Просто відкрийте файл `index.html` у вашому улюбленому браузері (Chrome, Firefox, Safari, Edge).

### Варіант 2: Запуск локального сервера розробника (NodeJS)
1. Встановіть залежності (якщо потрібно) та запустіть сервер:
   ```bash
   npm run dev
   ```
2. Перейдіть у браузері за адресою: [http://localhost:3000](http://localhost:3000).
